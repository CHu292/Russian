# Аннотирование научной статьи
(Ghi chú bài báo khoa học)


## Примерный план аннотации
Kế hoạch mẫu để viết ghi chú


| **Элемент аннотации** <br> (Thành phần ghi chú)                             | **Примеры выражений <br> (Ví dụ câu diễn đạt)**                                                                                                                                            |
|------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **1. Указание на содержание статьи / книги** <br> (Nêu nội dung của bài viết/cuốn sách)         | - В статье / книге / работе... *(Trong bài viết / cuốn sách / công trình...)*                                                                                                        |
|                                                      | - Говорится о / речь идёт о... *(Nói về / đề cập đến...)*                                                                                                                            |
|                                                      | - Затрагиваются / рассматриваются / обсуждаются вопросы / проблемы... *(Đề cập / xem xét / thảo luận các câu hỏi / vấn đề...)*                                                       |
|                                                      | - Статья посвящена (чему?) / анализу (чего?)... *(Bài viết dành cho (cái gì?) / phân tích (cái gì?)...)*                                                                              |
|                                                      | - В статье дан анализ (чего?) / рассматривается (что?) / характеризуется (что?)... *(Trong bài viết, phân tích (cái gì?) / xem xét (cái gì?) / được mô tả (cái gì?)...)*              |
|                                                      | - В статье / книге / работе исследуется / анализируется / даётся характеристика / раскрывается / подчёркивается... *(Trong bài viết / cuốn sách / công trình, nghiên cứu / phân tích / mô tả / trình bày / nhấn mạnh...)* |
| **2. Указание на особенности и наиболее важную информацию в статье / книге** <br> Nêu các đặc điểm và thông tin quan trọng nhất trong bài viết/cuốn sách | - Большое место в статье / книге / работе занимает рассмотрение (чего?)... *(Trong bài viết / cuốn sách / công trình, phần lớn dành để xem xét (cái gì?)...)*                        |
|                                                      | - Главное внимание в статье / книге / работе обращается на / особое внимание уделяется (чему?)... *(Sự chú ý chính trong bài viết / cuốn sách / công trình tập trung vào / đặc biệt chú ý đến (cái gì?)...)* |
| **3. Указание на авторскую позицию** <br> Nêu rõ quan điểm của tác giả                | - Автор статьи придерживается мнения о том, что / высказывает свою точку зрения на / затрагивает проблемы / рассматривает... *(Tác giả bài viết giữ quan điểm rằng / bày tỏ quan điểm của mình về / đề cập đến các vấn đề / xem xét...)* |
| **4. Указание на адресата**                          | - Статья / книга / работа адресована (кому?) / предназначена для (кого?) / может заинтересовать (кого?) / может быть полезна (кому?)... *(Bài viết / cuốn sách / công trình dành cho (ai?) / có thể quan tâm đến (ai?) / có thể hữu ích cho (ai?)...)* |
| **5. Указание на структуру**                         | - Статья / книга / работа состоит из двух / трёх частей... *(Bài viết / cuốn sách / công trình bao gồm hai / ba phần...)*                                                            |
|                                                      | - В первой части... *(Trong phần đầu tiên...)*                                                                                                                                       |
|                                                      | - Во второй части... *(Trong phần thứ hai...)*                                                                                                                                       |
|                                                      | - В заключительной части исследования кратко рассматривается (что?)... *(Trong phần cuối của nghiên cứu, ngắn gọn xem xét (cái gì?)...)*                                             |

## Языковые клише, которые используются при написании аннотаций
Các cụm từ cố định ngôn ngữ được sử dụng khi viết ghi chú.

# Активные и пассивные обороты речи  
*(Cách diễn đạt chủ động và bị động trong ngôn ngữ)*  

| **Активные обороты речи (Chủ động)**                                     | **Пассивные обороты речи (Bị động)**                                                                 |
|--------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| Автор рассматривает вопрос (проблему)... *(Tác giả xem xét câu hỏi (vấn đề)...)*                       | Рассматривается вопрос (проблема)... *(Câu hỏi (vấn đề) đang được xem xét...)*                     |
| Автор излагает вопрос (проблему, сущность)... *(Tác giả trình bày câu hỏi (vấn đề, bản chất)...)*      | Излагается вопрос (проблема, сущность)... *(Câu hỏi (vấn đề, bản chất) đang được trình bày...)*     |
| Автор освещает вопрос (проблему)... *(Tác giả làm sáng tỏ câu hỏi (vấn đề)...)*                       | Освещён вопрос (освещена проблема)... *(Câu hỏi (vấn đề) đã được làm sáng tỏ...)*                  |
| Автор останавливается на следующих вопросах... *(Tác giả tập trung vào các câu hỏi sau...)*           | Затронут вопрос (проблема)... *(Câu hỏi (vấn đề) đã được đề cập đến...)*                          |
| Автор затрагивает вопрос (проблему)... *(Tác giả đề cập đến câu hỏi (vấn đề)...)*                     | Описан метод (способ)... *(Phương pháp đã được mô tả...)*                                           |
| Автор описывает метод (способ)... *(Tác giả mô tả phương pháp (cách tiếp cận)...)*                   | Проведён анализ проблемы... *(Phân tích vấn đề đã được thực hiện...)*                              |
| Автор проводит анализ проблемы... *(Tác giả thực hiện phân tích vấn đề...)*                          | Дан анализ проблемы... *(Phân tích vấn đề đã được cung cấp...)*                                    |
| Автор показывает (раскрывает, анализирует, даёт анализ, исследует, описывает)... *(Tác giả trình bày (tiết lộ, phân tích, thực hiện phân tích, nghiên cứu, mô tả)...)* | Показан, раскрыт, проанализирован, дан анализ, даётся анализ, исследован, описан (-а, -о, -ы)... *(Được trình bày, tiết lộ, phân tích, cung cấp phân tích, nghiên cứu, mô tả...)* |
| Автор представляет обзор... *(Tác giả đưa ra cái nhìn tổng quan...)*                                | Представлен обзор... *(Cái nhìn tổng quan đã được trình bày...)*                                    |
| Особое внимание автор уделяет вопросу (проблеме)... *(Tác giả đặc biệt chú ý đến câu hỏi (vấn đề)...)* | Особое внимание уделяется вопросу (проблеме)... *(Câu hỏi (vấn đề) được đặc biệt chú ý...)*        |
| Автор заостряет внимание на... *(Tác giả nhấn mạnh vào...)*                                           | Особое внимание заострено на... *(Đã nhấn mạnh đặc biệt vào...)*                                   |
| Автор даёт характеристику (останавливается на характеристике)... *(Tác giả đưa ra đặc điểm (tập trung vào đặc điểm)...)* | Дана (представлена) характеристика... *(Đặc điểm đã được cung cấp (trình bày)...)*                |
| Автор подчёркивает важность... *(Tác giả nhấn mạnh tầm quan trọng...)*                              | Подчёркнута важность... *(Tầm quan trọng đã được nhấn mạnh...)*                                    |
| Автор указывает... *(Tác giả chỉ ra...)*                                                             | Указан (-а, -о, -ы)... *(Đã được chỉ ra...)*                                                       |
| Автор доказывает... *(Tác giả chứng minh...)*                                                        | Доказан (-а, -о, -ы)... *(Đã được chứng minh...)*                                                  |
| Автор приводит пример... *(Tác giả đưa ra ví dụ...)*                                                | Приведён (приведены) пример... *(Ví dụ đã được đưa ra...)*                                         |
| Автор представляет результаты... *(Tác giả trình bày kết quả...)*                                   | Представлены результаты... *(Kết quả đã được trình bày...)*                                        |
| Автор показывает значение... *(Tác giả chỉ ra ý nghĩa...)*                                          | Показано значение... *(Ý nghĩa đã được chỉ ra...)*                                                |
| Автор даёт оценку... *(Tác giả đưa ra đánh giá...)*                                                 | Дана оценка... *(Đánh giá đã được đưa ra...)*                                                     |
| Автор делает (формулирует) вывод... *(Tác giả đưa ra (hình thành) kết luận...)*                     | Сформулирован вывод... *(Kết luận đã được hình thành...)*                                         |
| Текст предназначен для... *(Văn bản được dành cho...)*                                              | Текст рассчитан на... *(Văn bản được thiết kế dành cho...)*                                        |

| приводить                | привести             | приведён                        | đưa ra                                            |
